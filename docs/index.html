This website will display the "Development" version of the ISO 14812 Concept Model for ITS Vocabulary. At present, there is a proposed version of the initial draft, which can be found at https://k-vaughn.github.io/iso14812/. If this draft is accepted by the working group as a starting point, it will be posted on this site. The decision to accept the proposal will be made during our next conference call scheduled for 11 Dec 2018.
